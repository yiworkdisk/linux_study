Beyond Here

  Environment
    OS : Windows10

Ubuntu-20.04(18.04 or later) 와 Windows10 멀티부팅 시간이 맞지 않음

Ubuntu-20.04(18.04 or later)은 UTC 사용하고 Windows10은 Local Time을 사용

Windows10을 UTC로 바꾸면 이 문제 해결 가능


1. Registry Editor 실행


Win+R 실행

regedit 실행

아래 경로로 이동

\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\TimeZoneInformation

New > DWORD (32bit) Value
RealTimeIsUniversal
Value data : 1
